# odin-tic-tac-toe
[Live Preview](https://norphel.github.io/odin-tic-tac-toe/)
A simple Tic-Tac-Toe game built to implement following concepts learned in previous lessons:
* Factory functions
* Closure
* Private variables and functions
* IIFE
* Module Pattern
* Namespacing
